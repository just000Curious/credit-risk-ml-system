# Credit Risk Modeling & Prediction System üè¶

## Overview
A machine learning-powered system that predicts loan default probability to help financial institutions make data-driven lending decisions.

## Key Features
- **Web Interface**: Professional dashboard with 3 tabs for workflow
- **Real-time Analysis**: Instant risk assessment as you input data
- **Predictive Model**: 92.4% accurate default probability prediction
- **Decision Support**: Clear approve/conditional/decline recommendations
- **Credit Scoring**: Generates credit scores (300-850) and risk ratings

## Quick Start


### Installation
git clone https://github.com/yourusername/credit-risk-modelling.git
cd credit-risk-modelling/app
pip install -r requirements.txt
streamlit run main.py

### Usage
1. **Tab 1**: Enter applicant details (age, income, loan amount, credit history)
2. **Click**: "Proceed to Risk Assessment" button
3. **Tab 2**: View risk analysis with default probability, credit score, and recommendations
4. **Tab 3**: Check model performance metrics

## What It Does
- Takes customer financial and loan information as input
- Calculates risk factors in real-time
- Predicts probability of loan default
- Suggests approval decisions with justifications
- Provides credit score and risk rating

## Input Parameters
- **Personal**: Age, Annual Income, Residence Type
- **Loan**: Amount, Tenure, Purpose, Type
- **Credit History**: Delinquency Rate, Credit Utilization, DPD
- **Additional**: Number of open accounts

## Output Results
- **Default Probability**: 0-100% with risk level (Low/Medium/High)
- **Credit Score**: 300-850 with category (Poor to Excellent)
- **Risk Rating**: AAA-C investment grade classification
- **Recommendation**: Approve, Conditional, or Decline with next steps

## Model Performance
- **Accuracy**: 92.4%
- **Precision**: 89.7%
- **Recall**: 87.3%
- **Training Data**: 58,421 samples

## Use Cases
- Banks & financial institutions for loan approval
- Credit analysts for risk assessment
- FinTech platforms for automated decisions
- Educational purposes for ML in finance

## Requirements
- Python 3.8+
- Streamlit and scikit-learn
- 4GB RAM minimum
- Modern web browser

## Deployment Options
- Local: Run with `streamlit run main.py`
- Cloud: Deploy to Streamlit Cloud, AWS, or Azure
- Docker: Containerize for production
